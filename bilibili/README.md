# *哔哩哔哩（bilibili | B站）多p 视频选集区标题显示加宽*

b站多p 视频选集的时候，遇到长标题有相同前缀占用太多内容，宽度不够。虽然鼠标悬停时可以看到完整标题，但是分p太多的时候，又要跳着看，这样仍然浪费时间，手也累，故开发此脚本，自动增加宽度。

常规播放、宽屏播放、网页全屏播放、全屏播放选集列表的宽度都可以自动增加
